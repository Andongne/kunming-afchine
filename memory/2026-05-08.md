# Session 2026-05-08 — Corrections CSS et incident SP Builder

## Travaux réalisés

### Harmonisation titres sidebar / CTA
- Problème : titres modules `rse-calendar-cta` non harmonisés avec sidebar
- Cause racine : CSS prod avait encore `color: #da002e` pour `.afk-sidebar-card .afk-sidebar-title` et aucune règle CTA
- Corrections CSS :
  - `.afk-sidebar-card .afk-sidebar-title` → `color: #1a171b` (noir)
  - `.afk-cta-row / .afk-cta-col` → `overflow: visible`
  - `.afk-cta-card` → `box-shadow: 0 2px 16px 0 rgba(0,0,0,0.12)`
  - `.afk-cta-card .afk-sidebar-title` → styles harmonisés
- Template `calendar/default.php` : titre CTA `<h3 style="...">` → `<div class="afk-sidebar-title">`

### Incident : version CSS Varnish stale
- Lors du bump `?v=2026050801` dans `index.php`, Varnish a mis en cache l'ancienne CSS sous la nouvelle URL
- Résolution : second bump `?v=2026050802` → Varnish MISS → bon CSS servi
- **Leçon : toujours déployer le CSS AVANT de bumper la version dans index.php**

- **Leçon : si sauvegarde SP Builder échoue silencieusement, vérifier `bwhwo_sppagebuilder_versions` et la taille du contenu**
