Reformatage des cards de la homepage de kunming-afchine.org pour la version mobile, inspiré d'un site concurrent (aisite.wejianzhan.com — style 2 colonnes, cards compactes type "bouton").

### Modifications CSS appliquées (mobile uniquement, via Helix Ultimate custom_css, style_id=10)

1. Fond blanc + bannière rouge + image — mais texte illisible, layout 1 colonne

3. (en cours) Gradient de fondu sur image, fond blanc, titre rouge `#DA002E`, hauteur uniforme via flexbox stretch

## Mise à jour session 06h15

### Problème critique résolu : CSS DB inaccessible au-delà de ~58KB SQL
- JSON_SET sur custom_css dans bwhwo_template_styles échoue avec 500 pour les payloads > ~58KB SQL
- Le CSS base (29007 chars) a été restauré dans la DB via JSON_SET (OK)
- Le bloc mobile cards (9.7KB) a été extrait dans un fichier dédié

### Architecture CSS finale adoptée
- **DB (custom_css)** : CSS base 29007 chars — desktop uniquement, styles globaux
- **Fichier SFTP** : `templates/shaper_languageschool/css/mobile_cards.css?v=20260429`
  - Chargé via `index.php` ligne 142 avec query string pour contourner Varnish
  - Contient tout le CSS mobile des 6 cards homepage

### Pour toute modification CSS mobile future
Modifier `/tmp/mobile_cards.css` localement → upload SFTP → changer le numéro de version dans index.php

### Desktop : INTACT (ne jamais toucher custom_css DB au-delà de 29007 chars)
