### RSForm Pro — Formulaire d'inscription (FormId=4)

#### Problème JS dans le head sans `<script>` tags
- **Cause** : RSForm Pro utilise `addCustomTag()` pour le champ `JS`, pas `addScriptDeclaration()` → le contenu est injecté tel quel dans `<head>` sans balises
- **Fix** : Ajouter `<script>` et `</script>` autour du code JS dans le champ `bwhwo_rsform_forms.JS`
- **Cache** : Le cache Joomla doit être purgé après chaque modification DB (via `sppb5.php?action=purge_cache`)
- **Table prefix** : `bwhwo_` (pas `jos_`)
- **select_query** : utilise le paramètre `sql` (pas `q` ou `query`)
- **write_query** : utilise UNHEX() pour les valeurs avec caractères spéciaux

#### Bouton "Continue" post-soumission masqué
- CSS ajouté dans `bwhwo_rsform_forms.CSS` (FormId=4) : `.rsform-block-continue { display: none !important; }`
